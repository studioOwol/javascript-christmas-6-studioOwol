## **🚀 구현 기능 목록**

### 메뉴 종류 구분(menus)

- [ ] 판매중인 메뉴의 종류와 가격 정보를 저장한다.

### 주문(Order)

- [ ] 고객이 주문한 메뉴의 이름과 개수가 담긴 주문 내역을 만든다.
- [ ] 총주문 금액을 계산한다.
- [ ] 이벤트에 적용되는 메뉴 종류의 개수를 구한다.
- [ ] 예외 처리
  - [ ] 주문 개수가 1 이하인 경우
  - [ ] 총 주문 개수가 20개 이상인 경우
  - [ ] 메뉴판에 없는 메뉴인 경우
  - [ ] 주문 형식에 맞지 않는 경우
  - [ ] 중복 메뉴가 있는 경우
  - [ ] 음료만 주문하는 경우
